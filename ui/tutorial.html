KATch2 NetKAT Tutorial

This tutorial will guide you through the basics of NetKAT.

Explain that the packet is a vector of bits. Explain that variables are x0, x1, x2, etc. Represents the n-th bit of the packet. Explain tests: x3==0, x3==1 (packet filters) Explain sequential composition: x1==0; x2==0. Explain assignments: x0:=0; x1:=1. Explain choice: x0:=1 + x1:=1. Explain how to encode if then else using tests and choice. Explain iteration: e*. Give some nice examples. Explain dup: logs the current packet to the trace. Give examples.
