NetKAT Interactive Tutorial

This page demonstrates how the NetKAT editor library can be used to create interactive tutorials. The library automatically transforms <pre class="netkat"> elements and <netkat-editor> custom elements into live editors with real-time analysis.

1. Basic NetKAT Expressions

Let's start with a simple field assignment. Try editing the code below:

x0 := 0
Note: This expression sets field x0 to 0. The analysis should show whether this expression is satisfiable.

2. Sequential Composition

NetKAT uses semicolon (;) for sequential composition. Here's an example that sets two fields:

x0 := 0; x1 := 1

Try modifying the values or adding more field assignments.

3. Conditional Expressions

You can use tests and branches to create conditional behavior:

x0 == 0; x1 := 1 + x0 == 1; x1 := 0
Explanation: This expression uses the plus operator (+) for choice. If x0 equals 0, then set x1 to 1. Otherwise, if x0 equals 1, set x1 to 0.

4. Star Operation (Iteration)

The star operator (*) represents iteration. Here's a more complex example:

x0 := 0; x1 := 0;
((x0 == 0; x0 := 1 + x0 == 1; x1 := 1); dup)*;
x0 == 0; x1 == 1

This example demonstrates a state machine that alternates between states.

5. Using Custom Elements

You can also use custom <netkat-editor> elements:

// This is a custom element example // Try adding some NetKAT code here dup
dup: The dup operator duplicates packets, which is useful for broadcasting.

6. Error Handling

Try introducing a syntax error to see how the editor handles it:

x0 := // This will cause a syntax error

The editor will highlight syntax errors and show helpful error messages.

7. Empty Expressions

Some NetKAT expressions might be unsatisfiable (empty). Try this one:

x0 := 0; x0 := 1
Analysis: This expression is contradictory - it's impossible to set x0 to both 0 and 1 in sequence.

How to Use This Library

To use this NetKAT editor library in your own tutorials:

  1. Include the netkat-editor.js script in your HTML
  2. Make sure the WASM files are accessible at ./pkg/
  3. Add <pre class="netkat"> elements with your NetKAT code
  4. Or use <netkat-editor> custom elements
  5. The library will automatically transform them into interactive editors!
